# 포트폴리오 타입스크립트로 변환

js로 되어있던 포트폴리오를 ts로 바꿔보려고 한다.
언제나 그렇지만 과거에 작성한 코드를 보면 다른 사람이 만든 거 같다.
바꾸는 김에 이것 저것 넣어보고 싶었던 기능도 추가해보려고 한다.
물론 안될 수도 있다.

## 나름 목표

### ts 공부

회사에서 쓰고는 있는데 구현에 급급해서 잘 쓰고 있는지 모르겠다.
천천히 바꾸면서 ts를 씹고 뜯고 맛보고 즐기려고 한다.

### 새로운 라이브러리 사용

써보고 싶은 라이브러리가 계속 생긴다.
매번 사용하던 redux를 뺴고 recoil을 써보려고한다.
서버 상태는 react-query를 쓴다.

### 기능 추가

저번 포트폴리오를 만들 떄는 생각만 했던 기능들을 넣어볼까 한다.
컨셉이 os니까 파일 시스템 같은 걸 구현해볼까
아니면 cmd창을 만들어볼까 휴지통을 넣어볼까
물론 못할 수도 있다.

### 문서화

처음부터 끝까지 문서화하면서 개발을 마무리해보고 싶다.
기획부터 엑셀이나 워드로 정리하고
디자인은 피그마로, 개발은 기능별로 브랜치를 정리해서 깔끔하게,
물론 안할 수도 있다.
